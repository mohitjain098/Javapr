9. How to convert String to Date

Solution


    // Conversion Example
    import java.text.SimpleDateFormat;
    import java.util.Date;
    
    String str = "2024-11-03";
    SimpleDateFormat formatter = new SimpleDateFormat("yyyy-MM-dd");
    Date result = formatter.parse(str);
    System.out.println(result);
Output: Sun Nov 03 00:00:00 UTC 2024

10. How to convert Date to String

Solution


    // Conversion Example
    import java.text.SimpleDateFormat;
    import java.util.Date;
    
    Date date = new Date();
    SimpleDateFormat formatter = new SimpleDateFormat("yyyy-MM-dd");
    String result = formatter.format(date);
    System.out.println(result);
Output: "2024-11-03"

23. How to convert Date to Timestamp

Solution


    // Conversion Example
    import java.util.Date;
    import java.sql.Timestamp;
    
    Date date = new Date();
    Timestamp result = new Timestamp(date.getTime());
    System.out.println(result);
Output: Timestamp value of current date and time

24. How to convert Timestamp to Date

Solution


    // Conversion Example
    import java.sql.Timestamp;
    import java.util.Date;
    
    Timestamp timestamp = new Timestamp(System.currentTimeMillis());
    Date result = new Date(timestamp.getTime());
    System.out.println(result);
Output: Date value of the timestamp
