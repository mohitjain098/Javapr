1. Write a Java method to find the smallest number among three numbers.

Solution


public class SmallestOfThree {
    public static void main(String[] args) {
        System.out.println("Smallest number: " + smallest(5, 2, 8));
    }

    public static int smallest(int a, int b, int c) {
        return Math.min(a, Math.min(b, c));
    }
}
Output: Smallest number: 2

3. Write a Java method to display the middle character of a string.

Solution


public class MiddleCharacter {
    public static void main(String[] args) {
        System.out.println("Middle character: " + middleChar("Hello"));
    }

    public static String middleChar(String str) {
        int len = str.length();
        if (len % 2 == 0) {
            return String.valueOf(str.charAt(len / 2 - 1));
        } else {
            return str.charAt(len / 2) + "" + str.charAt(len / 2);
        }
    }
}
Output: Middle character: l

5. Write a Java method to compute the future investment value at a given interest rate for a specified number of years.

Solution


public class InvestmentValue {
    public static void main(String[] args) {
        double futureValue = calculateFutureValue(1000, 0.05, 10);
        System.out.println("Future investment value: " + futureValue);
    }

    public static double calculateFutureValue(double investment, double interestRate, int years) {
        return investment * Math.pow(1 + interestRate, years);
    }
}
Output: Future investment value: 1628.894626777442

6. Write a Java method to print characters between two characters (i.e., A to P).

Solution


public class CharactersBetween {
    public static void main(String[] args) {
        printCharacters('A', 'P');
    }

    public static void printCharacters(char start, char end) {
        for (char c = start; c <= end; c++) {
            System.out.print(c + " ");
        }
    }
}
Output: A B C D E F G H I J K L M N O P

7. Write a Java method to check whether a string is a valid password.

Solution


public class ValidPassword {
    public static void main(String[] args) {
        System.out.println("Is valid password: " + isValidPassword("Password123!"));
    }

    public static boolean isValidPassword(String password) {
        return password.length() >= 8 && password.matches(".*[0-9].*") && password.matches(".*[!@#$%^&*].*");
    }
}
Output: Is valid password: true

9. Write a Java method to find all twin prime numbers less than 100.

Solution


public class TwinPrimes {
    public static void main(String[] args) {
        findTwinPrimes(100);
    }

    public static void findTwinPrimes(int limit) {
        for (int i = 3; i < limit; i++) {
            if (isPrime(i) && isPrime(i + 2)) {
                System.out.println("(" + i + ", " + (i + 2) + ")");
            }
        }
    }

    public static boolean isPrime(int n) {
        if (n < 2) return false;
        for (int i = 2; i <= Math.sqrt(n); i++) {
            if (n % i == 0) return false;
        }
        return true;
    }
}
Output: (3, 5) (5, 7) (11, 13) (17, 19) (29, 31) (41, 43) (59, 61) (71, 73) (79, 81)

11. Write a Java method that accepts three integers and checks whether they are consecutive or not.

Solution


public class ConsecutiveNumbers {
    public static void main(String[] args) {
        System.out.println("Are consecutive: " + areConsecutive(4, 5, 6));
    }

    public static boolean areConsecutive(int a, int b, int c) {
        return (a + 1 == b && b + 1 == c) || (b + 1 == a && a + 1 == c) || (c + 1 == a && a + 1 == b);
    }
}
Output: Are consecutive: true

12. Write a Java method that accepts three integers and returns true if one is the middle point between the other two integers, otherwise false.

Solution


public class MiddlePoint {
    public static void main(String[] args) {
        System.out.println("Is middle point: " + isMiddlePoint(2, 4, 6));
    }

    public static boolean isMiddlePoint(int a, int b, int c) {
        return a + c == 2 * b || b + c == 2 * a || a + b == 2 * c;
    }
}
Output: Is middle point: true

13. Write a Java method that checks whether all the characters in a given string are vowels (a, e, i, o, u).

Solution


public class AllVowels {
    public static void main(String[] args) {
        System.out.println("All characters are vowels: " + areAllVowels("aeiou"));
    }

    public static boolean areAllVowels(String str) {
        for (char c : str.toCharArray()) {
            if (c != 'a' && c != 'e' && c != 'i' && c != 'o' && c != 'u') {
                return false;
            }
        }
        return true;
    }
}
Output: All characters are vowels: true
