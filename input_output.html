Java Applet Examples

1. Enter the character from user using InputStreamReader

Solution


import java.applet.Applet;
import java.awt.*;
import java.io.*;

public class InputCharacterApplet extends Applet {
    String input;

    public void init() {
        try {
            BufferedReader reader = new BufferedReader(new InputStreamReader(System.in));
            input = "Enter a character: " + (char) reader.read();
        } catch (IOException e) {
            input = "Error reading input.";
        }
    }

    public void paint(Graphics g) {
        g.drawString(input, 20, 20);
    }
}
Output: Displays the entered character on the applet.
Example: If the user enters 'A', the output will be: "Enter a character: A"

9. Get all characters from the file and display on the screen

Solution


import java.applet.Applet;
import java.awt.*;
import java.io.*;

public class FileReadApplet extends Applet {
    StringBuilder input = new StringBuilder();

    public void init() {
        try (BufferedReader reader = new BufferedReader(new FileReader("input.txt"))) {
            String line;
            while ((line = reader.readLine()) != null) {
                input.append(line).append("\n");
            }
        } catch (IOException e) {
            input.append("Error reading file.");
        }
    }

    public void paint(Graphics g) {
        g.drawString(input.toString(), 20, 20);
    }
}
Output: Displays all characters from the file on the applet.
Example: If "input.txt" contains:
``` Hello World ``` The output will be:
"Hello
World"

10. Get all characters from the file, count number of lines, words, characters and display on the screen

Solution


import java.applet.Applet;
import java.awt.*;
import java.io.*;

public class FileStatsApplet extends Applet {
    StringBuilder input = new StringBuilder();
    int lineCount = 0, wordCount = 0, charCount = 0;

    public void init() {
        try (BufferedReader reader = new BufferedReader(new FileReader("input.txt"))) {
            String line;
            while ((line = reader.readLine()) != null) {
                input.append(line).append("\n");
                lineCount++;
                wordCount += line.split("\\s+").length;
                charCount += line.length();
            }
        } catch (IOException e) {
            input.append("Error reading file.");
        }
    }

    public void paint(Graphics g) {
        g.drawString(input.toString(), 20, 20);
        g.drawString("Lines: " + lineCount, 20, 80);
        g.drawString("Words: " + wordCount, 20, 100);
        g.drawString("Characters: " + charCount, 20, 120);
    }
}
Output: Displays all characters, along with line, word, and character counts from the file.
Example: If "input.txt" contains:
``` Hello World Welcome to Java ``` The output will be:
"Hello World
Welcome to Java
Lines: 2
Words: 5
Characters: 30"
